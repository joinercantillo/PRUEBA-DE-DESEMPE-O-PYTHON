The Program is a simple sistem to manage data of all students at the SCHOOL ****, is easy to use:

with that in the begin you can see the main menu, you can use this options:
1 - add students using the ID student, name, age, grade and state

2 - list students: if you wanna see all the students you have in database

3 - delete students: if you need delete any student of your data base you can do it using the ID of this student.

4 - search student: when you need search any student in your data base please chose the option 4 and enter de ID of this student

5 - update student: if you need update the student data, please use the 5 button and update all the student data, if you keep any item only please "enter" button

6 - exit: if you wanna finish the process enter de "6" button a have a good day.

all the data is conserved in the csv file called "student.csv" any time, when you start the program or finish this all items keep in this file.
